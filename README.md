# GMITO: The Monster Hunter

Projeto Godot 4.6.3 criado a partir do documento `Game Design.pdf`.

## Como abrir

Abra a pasta `gmito-monster-hunter` pelo Godot ou execute:

```sh
/Users/higorwilvert/Downloads/Godot.app/Contents/MacOS/Godot --path /Users/higorwilvert/Documents/Faculdade/GAMEDEV/gmito-monster-hunter
```

## Controles

- A/D ou setas: mover
- Espaco: pular e pulo duplo
- Shift: dash energetico
- Esc: pausar

## Conteudo

- 3 fases: Lua Acucarada, Planeta Vulcanico e Planeta Congelado
- Doces cosmicos, mini energeticos, coracoes e latas principais
- Checkpoints, inimigos terrestres, inimigos voadores, lava, espinhos de gelo e plataformas moveis
- Chefe final Dr. Vazio com 3 acertos e projeteis
- Menu, HUD, pausa e cutscenes simples
